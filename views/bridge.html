QLC+ Bridge

This page listens to socket events from the beer-fest.js server just as any other client would, it then translates those messages to native websockets messages for QLC+.

To use this bridge, open QLC+. with the web interface enabled by running qlcplus -w, load up qlcplus.qxw, then refresh this page.

Have a look at the JavaScript source code for this page to see how it works.
